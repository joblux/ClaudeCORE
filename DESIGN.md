# JOBLUX Design System

## Brand Direction
Luxury, executive, discreet, premium, editorial confidence.
Never flashy. Never mass-market SaaS. Never playful startup UI.

## Core Colors
Background: #1a1a1a
Surface: #222222
Border: #333333
Primary Text: #ffffff
Secondary Text: #bdbdbd
Accent Gold: #a58e28
Success: restrained green
Error: restrained red

## Typography
Headlines: Playfair Display
UI / Body: Inter

Use larger elegant headings, restrained body text.

## Spacing & Shape
Generous whitespace.
Cards with clean radius.
No cramped layouts.
No excessive shadows.

## Components
Buttons: premium, calm, confident.
Inputs: dark, clean, high contrast.
Cards: elegant, readable, dense but breathable.
Tables: executive dashboard style.

## UX Rules
Clarity over cleverness.
Prestige over trendiness.
Fast comprehension.
Desktop-first if scope unclear.
Mobile polish can follow.

## Forbidden
No bright startup gradients.
No cartoon icons.
No neon effects.
No clutter.
No redesign drift without explicit approval.

## Agent Rule
Any new UI must align with DESIGN.md before implementation.
